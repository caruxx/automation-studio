プライバシーポリシー

事業者: Caruvi Star / 所在地: 日本

1. 所在地

本サービスの運営者は Caruvi Star、所在地は日本です。

2. YouTube API Services の使用

本サービスは YouTube API Services を使用し、チャンネル、動画、統計、サムネイル等の情報を取得・更新する場合があります。

3. Google プライバシーポリシー

Google の取り扱いは Google プライバシーポリシー をご確認ください。

4. 収集・保存・共有するデータ

OAuth トークン、チャンネル情報、動画メタデータ、公開統計、分析キャッシュ、操作ログをローカルまたは運用環境に保存します。第三者への販売や広告目的の共有は行いません。

5. 第三者広告

本サービスは第三者広告を表示せず、広告配信目的で YouTube API 由来データを共有しません。

6. デバイス情報・Cookie

ローカルダッシュボードの認証維持や操作性のため、ブラウザの Cookie / localStorage を使用する場合があります。端末識別広告には使用しません。

7. セキュリティ設定と削除

Google アカウント連携は Google セキュリティ設定 から確認・解除できます。本サービス上の「YouTube連携解除」でもトークン失効と API 由来データ削除を行えます。

Privacy Policy

Operator: Caruvi Star / Location: Japan

1. Location

This service is operated by Caruvi Star in Japan.

2. Use of YouTube API Services

This service uses YouTube API Services to access or update channel, video, statistics, and thumbnail data.

3. Google Privacy Policy

Please review the Google Privacy Policy.

4. Data Collected, Stored, and Shared

The service may store OAuth tokens, channel information, video metadata, public statistics, analysis caches, and audit logs. It does not sell this data or share it for advertising.

5. No Third-Party Advertising

The service does not display third-party ads or share YouTube API-derived data for ad serving.

6. Device Information and Cookies

The local dashboard may use cookies or localStorage for authentication and usability. These are not used for device-targeted advertising.

7. Security Settings and Deletion

You can manage Google account access in Google Security Settings. The service also provides a YouTube disconnect action that revokes tokens and deletes API-derived data.
