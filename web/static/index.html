RA
外出先のスマホからダッシュボードを操作できるようにする設定です。下の3ステップを順番に実行してください。「コピー」ボタンでコマンドをクリップボードへコピーできます。
1
ターミナルでトンネルを起動
Mac の「ターミナル」アプリを開き、下記コマンドを貼り付けて実行すると https://xxxxx.trycloudflare.com のような公開 URL が表示されます。その URL を次の欄に貼り付けます。
bash Python/setup_tunnel.sh
「ターミナル」は Launchpad → その他 → ターミナル で開けます。このウインドウは閉じずに開いたままにしてください。
2
表示された URL をここに保存
ステップ1で表示された trycloudflare.com の URL を貼り付けて「URL を保存」を押してください。
3
スマホからアクセス
下のアクセストークンをスマホのログイン画面で入力してください。QR コードを読み取ると URL を手入力せずに開けます。
再生成すると既存のスマホセッションは自動でログアウトされます。
認証を必須にしたい（パスワード保護を追加）
既定ではトークン入力なしでもアクセスできます。ログインを必須にするには、サーバーを以下のコマンドで起動し直してください。
ORZZ_AUTH_REQUIRED=1 bash Python/start.sh
設定のバックアップ / 復元
すべての設定（チャンネル情報・SUNO・ベンチマーク・プロンプト・スケジュールなど）を1つの JSON ファイルに書き出せます。別のパソコンへ移すときや、設定前の状態に戻したいときに使えます。
